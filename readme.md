# nonogram
https://en.wikipedia.org/wiki/Nonogram

## keyboard
* q: mark ?(temporal white) at current cell
* w: mark X(white)
* e: mark *(temporal black)
* r: mark #(black)
* t: empty current cell
* a: paint clues green for matched line
* s: mark trivial cells
* z: clean all marks
* 1: mark ? when clicked
* w: mark X when clicked
* e: mark * when clicked
* r: mark # when clicked
* t: empty current cell when clicked

## format
* cols: number of cols
* rows: number of rows
* chints: clues on the top, separated numbers by space    
* rhints: clues on the left, seperated numbers by space
